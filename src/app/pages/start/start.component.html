Test applicatie voor DevOps

Deze applicatie oplossing is voornamelijk bedoeld om Dela Azure te kunnen testen op de System team Azure OTAP-straat en dan voornamelijk de deployment pipelines. Het nieuwe Dela System team zal zich voornamelijk bezighouden met DevOps werkzaamheden.

Hierbij is het automatiseren van het automatiseren een hoog over doel. Deze applicatie is specifiek bedoelt om de verschillende typen pipelines in Azure DevOps te kunnen testen en ontwikkelingen te kunnen maken, waarin verschillende zaken samen komen.

Deze applicatie biedt CRUD operaties en kan ook direct benaderd worden via REST API calls. Ook wordt gewerkt aan o.a.: automatische deployments, automatische testen, quality gates en observability.
